## Bugs ##

# Turn
Round timer can be switched even though a projectile is active.

Player can still interact, jump and move, after firing a projectile.

# Weapons
All weapons have the same speed.

Melee shoots like a bullet.

Grenade has no radial damage.

# Terrain
When character jumps on bottom of terrain user sets to grounded.

Hitbox on edges of terrain need to be manually edited.

# Character
Character hitbox is very thin - which may give projectile hits a weird feel.

# UI
Cooldowns are not displayed on attack icons.
